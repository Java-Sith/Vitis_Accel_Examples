Prerequisites
=============

Vitis Accel Examples assumes that user is familiar with basic Vitis Environment, Setup, Programming and Debugging Flow. If not, it is recommended to cover these topics from Vitis Application Acceleration Guide:

    * `Vitis Application Acceleration Guide`_

.. _Vitis Application Acceleration Guide: https://www.xilinx.com/support/documentation/sw_manuals/xilinx2019_1/ug1393-vitis-application-acceleration.pdf
